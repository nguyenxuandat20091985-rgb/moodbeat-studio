## ✨ Tính năng nổi bật

- 🤖 **AI Enhancement** - Tự động phân tích ảnh, tối ưu màu sắc và độ sáng
- 🎵 **Beat Sync** - Đồng bộ ảnh với nhạc theo nhịp, tạo hiệu ứng chuyển cảnh ấn tượng
- 📱 **Multi-Platform** - Hỗ trợ TikTok, Reels, YouTube, Instagram, Shorts
- ⚡ **Lightning Fast** - Xử lý nhanh với FFmpeg.wasm, không cần tải lên server
- 🔒 **Private & Secure** - 100% xử lý trên trình duyệt, dữ liệu không bao giờ rời khỏi máy bạn
- 🎬 **Preview Real-time** - Xem trước video trước khi tải xuống
- 📦 **Đa định dạng** - Hỗ trợ JPG, PNG, MP3, WAV, MP4, MOV

## 📦 Cài đặt & Chạy local

### Yêu cầu
- Node.js (phiên bản 16+)
- Trình duyệt hiện đại (Chrome, Firefox, Safari, Edge)

### Các bước

```bash
# 1. Clone repository
git clone https://github.com/yourusername/moodbeat-studio.git
cd moodbeat-studio

# 2. Cài dependencies (optional)
npm install

# 3. Chạy local server
npm start
# hoặc
npx serve .

# 4. Mở trình duyệt tại: http://localhost:3000
