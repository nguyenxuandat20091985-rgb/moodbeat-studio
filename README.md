# 🎵 MoodBeat Studio

> Công cụ xử lý video chuyên nghiệp với AI, hỗ trợ đa nền tảng TikTok, Reels, YouTube và Facebook.

[![Vercel Deploy](https://img.shields.io/badge/Vercel-Deployed-000?style=for-the-badge&logo=vercel)](https://moodbeat-studio.vercel.app)
[![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)](LICENSE)

## ✨ Tính năng nổi bật

- 🤖 **AI Enhancement** - Tự động nâng cao chất lượng video
- 🎵 **Beat Sync** - Đồng bộ video với nhạc theo nhịp
- 📱 **Multi-Platform** - Hỗ trợ TikTok, Reels, YouTube, Facebook
- ⚡ **Lightning Fast** - Xử lý nhanh với FFmpeg.wasm
- 🔒 **Private & Secure** - Xử lý hoàn toàn trên trình duyệt

## 🚀 Cài đặt

```bash
# Clone repository
git clone https://github.com/yourusername/moodbeat-studio.git

# Di chuyển vào thư mục
cd moodbeat-studio

# Mở index.html trên trình duyệt
